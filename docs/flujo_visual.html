Ecosistema BrunoBot

Arquitectura de Auditoría e Inventario Inteligente

📧

1. Origen: Gmail

El POS envía automáticamente los cierres de caja y ventas en PDF. El banco envía las liquidaciones BAC/BANPRO.

Input de Datos
🧠

2. Proceso: Gmail Monitor

Un servicio 24/7 que escucha nuevos correos, descarga los PDFs y filtra la información relevante.

Script: gmail_monitor.py
📊

3. Auditoría: Análisis PDF

El cerebro extrae datos numéricos de los PDFs, calcula faltantes de efectivo y cruza ventas reales contra el POS.

Script: audit_cierre.py
🗄️

4. Almacén: PostgreSQL

Todo se guarda de forma estructurada para análisis histórico y visualización en dashboards (Metabase).

Persistencia
📱

5. Alerta: Telegram

Bruno envía un resumen ejecutivo al grupo de administración: Ventas, Faltantes, Top Productos y Alertas de Stock.

Notificación Real-Time
